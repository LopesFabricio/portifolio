SOBRE MIM

Olá, me chamo Fabrício e tenho 21 anos. Sou natural do Maranhão porém até os meus 19 anos cresci e estudei em Guarulhos-SP. Aos 19 anos (2020) me mudei para o interior do Maranhão, para uma outra realidade e em uma cidade que comporta por volta de 14 mil habitantes. Conheci a programação justamente pelo mercado de trabalho da minha cidade não ser muito favorável. Comecei pesquisar formas de trabalhar pela internet, quando encontrei o mundo da programação foi amor a primeira vista. Comecei meus estudos na programação em 2022. Gosto de trabalhar em equipe, tenho vontade de aprender e sempre tento me superar a cada novo passo. Acredito que com foco, determinação e esforço tudo é possivel. Vamos trabalhar juntos?

Minhas Skills

JavaScript

JavaScript é uma linguagem de programação de alto nível criada, a princípio, para ser executada em navegadores e manipular comportamentos de páginas web.

TypeScript

TypeScript é um superconjunto de JavaScript, ou seja, um conjunto de ferramentas e formas mais eficientes de escrever código JavaScript, adicionando recursos que não estão presentes de maneira nativa na linguagem.

HTML 5

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

CSS 3

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

React

O React é a biblioteca mais popular do JavaScript e é usada para construir uma interface de usuário (IU). Ela oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.

Bootstrap

Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.
